# MEE Publication Gate Review Confirmation Policy V1

Status: plan only

## Objective

Define the deterministic internal policy for deciding when a publication-gate `defer_review_confirmation` row may be advanced with `confirm_internal_candidate`.

This policy does not publish prices. It only describes when an internal review disposition may be considered for a future controlled function invocation package.

## Inputs

- `public.v_market_evidence_publication_gate_candidates_v1`
- `public.market_evidence_review_dispositions`
- `public.market_listing_card_candidates`
- `public.market_listing_observations`
- Existing action function contract for `public.apply_market_evidence_review_action_v1`
- Audit manifest: `docs/audits/market_evidence_engine_v1/MEE-PUBLICATION-GATE-REVIEW-ACTION-PLAN-V1/defer_review_confirmation_manifest.json`

## Hard Requirements

A row is not eligible for `confirm_internal_candidate` unless all requirements are true:

- `gate_decision = 'defer_review_confirmation'`
- `evidence_lane in ('raw_single', 'slab')`
- `review_lane in ('candidate_review', 'high_signal_review')`
- `review_status in ('pending', 'in_review')`
- `can_publish_price_directly = false`
- `publishable = false`
- `app_visible = false`
- `market_truth = false`
- no candidate rows for the card/lane are still marked `needs_review`
- no candidate rows for the card/lane have exclusion flags
- no candidate rows have `can_publish_price_directly = true`
- minimum candidate match confidence is at least `0.90`
- latest evidence is not stale for the lane

If any hard requirement fails, the row remains a policy hold.

## Lane Thresholds

### raw_single

- minimum matched listing count: `5`
- minimum distinct seller count: `3`
- stale after: `14 days`
- minimum match confidence: `0.90`
- high-value manual-review hold: `p95_total_ask_price >= 250`

### slab

- minimum matched listing count: `3`
- minimum distinct seller count: `2`
- stale after: `30 days`
- minimum match confidence: `0.90`
- high-value manual-review hold: `p95_total_ask_price >= 500`

## Special-Lane Holds

The following lanes are held for manual review even when basic volume and confidence thresholds pass:

- World Championship Deck replicas
- McDonald's collection cards
- MEP Black Star Promos
- Trainer Kit rows
- Base Set print-run lanes
- other promo/special-lane identities where the ordinary-card source image or ordinary-card listing text can be misleading

- `378` raw single rows: `0` confirmable
- `92` slab rows: `0` confirmable

- public-boundary leak rows: `0`

Primary hold reasons:

- `470` match confidence below policy
- `470` candidate rows still need review

- `60` exclusion flags present
- `15` insufficient seller diversity
- `14` insufficient listing count

## Decision

No `confirm_internal_candidate` apply package is generated by this policy package.

The publication gate has near-ready internal candidates, but the underlying candidate evidence still needs deterministic normalization/review before any row can be advanced. This is the correct stop point: the engine now has a publication gate and a deterministic confirmation policy, but it does not silently convert active listings into app-visible price truth.

## Boundary

This policy cannot:

- create public pricing
- write `pricing_observations`
- write `ebay_active_prices_latest`
- mark evidence as market truth
- mark rows app-visible
- mutate identity, vault, image, or card-print tables
- bypass the review action function
